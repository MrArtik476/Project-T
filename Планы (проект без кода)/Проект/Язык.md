Основной язык разработки: C# (стандарт для Unity)

Что нужно знать/выучить:
- Базовый синтаксис C# (переменные, классы, функции, ООП)
- Работа с MonoBehaviour (жизненный цикл объектов Unity — Start, Update и т.д.)
- Корутины (Coroutines) — для анимаций переходов, таймеров кризисов
- Основы работы с сетевым фреймворком (см. [[Техническая часть]]) — там свой слой синтаксиса поверх C#
